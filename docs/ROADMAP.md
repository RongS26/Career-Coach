# Roadmap / 路线图

[English](#english) | [中文](#中文)

## English

### V1: Local Markdown Coach

- Define generic context schema
- Add workflow templates
- Add system prompt / skill entry
- Run in an agent tool or local script
- Keep all personal context out of git

### V2: Local Web App

- Add simple chat UI
- Add workflow buttons
- Support multiple model providers
- Load local context files
- Save local-only session history

### V3: Knowledge Base

- Add curated reference digests
- Add optional retrieval over long reference notes
- Add source labels to recommendations
- Add prompt tests for generic-advice regression

### V4: Personal Learning Loop

- Save sanitized weekly insights
- Track repeated blockers and strengths
- Generate monthly review summaries
- Suggest learning projects based on recurring gaps

### V5: Privacy And Portability

- Add redaction checks
- Add encrypted local storage
- Add export/import
- Add data deletion flow
- Add configuration profiles for different roles

### V6: Hosted Multi-User Version

- Add authentication
- Add per-user encrypted storage
- Add billing and usage limits
- Add admin-free data deletion
- Add compliance review for sensitive career data

## 中文

### V1：本地 Markdown Coach

- 定义通用 context schema
- 添加 workflow templates
- 添加 system prompt / skill entry
- 在 agent 工具或本地脚本中运行
- 保持所有个人上下文不进 git

### V2：本地 Web App

- 添加简单 chat UI
- 添加 workflow buttons
- 支持多个模型 provider
- 加载本地 context files
- 保存仅本地可见的 session history

### V3：知识库

- 添加 curated reference digests
- 对长 reference notes 增加可选 retrieval
- 给 recommendations 添加 source labels
- 增加 generic-advice regression 的 prompt tests

### V4：个人学习闭环

- 保存已脱敏的 weekly insights
- 跟踪重复出现的 blockers 和 strengths
- 生成 monthly review summaries
- 根据反复出现的缺口建议 learning projects

### V5：隐私与可迁移性

- 添加 redaction checks
- 添加 encrypted local storage
- 添加 export/import
- 添加 data deletion flow
- 为不同角色添加 configuration profiles

### V6：托管多用户版本

- 添加 authentication
- 添加 per-user encrypted storage
- 添加 billing 和 usage limits
- 添加无需管理员介入的数据删除能力
- 针对敏感职业数据做 compliance review
